# DermaNerd

A modern skincare discovery platform built with React and Vite.

DermaNerd helps users understand their skin through personalized quizzes, AI-powered skincare assistance, and curated skincare recommendations.

---

## Features

- Modern responsive UI
- Animated hero section
- Personalized skincare quiz
- AI skincare assistant section
- Contact form section
- Mobile responsive navbar
- Glassmorphism design
- Smooth animations with Framer Motion

---

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
